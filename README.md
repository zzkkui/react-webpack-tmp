## react-webpack-tmp

### 开发环境

`node 10+`

#### 安装

``` bash
yarn or npm install
```

#### 运行

// 本地开发命令

``` bash
yarn start
```

// 本地mock

``` bash
yarn mock
```

// 打包上线命令

``` bash
yarn build
```
